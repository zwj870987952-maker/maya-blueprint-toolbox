# 类型 1：常量

## 声明

常量节点只提供固定数据。

它不读取 Maya 场景。
它不修改 Maya 场景。
它没有执行副作用。
它的输出值只由节点自身参数决定。

## 节点颜色

`#4EA1FF`

## 允许行为

- 提供文本
- 提供数字
- 提供布尔值
- 提供路径

## 禁止行为

- 不能读取 Maya 当前选择
- 不能检查 Maya 节点是否存在
- 不能获取或设置 Maya 属性
- 不能创建、删除、重命名对象
- 不能导入或导出文件

## 通用端口规则

输入端口：无

输出端口：一个主输出，通常叫“值”

## 当前节点

| 节点 | 输出端口 | 参数 |
| --- | --- | --- |
| 文本 | `值: STRING` | `值: text` |
| 数字 | `值: NUMBER` | `值: number` |
| 布尔 | `值: BOOL` | `值: bool` |
| 路径 | `值: PATH` | `路径: path` |

## 设计备注

文本、数字、布尔、路径都可以是用户手写输入，但输出语义不同。

例如用户写 `123`：

- 文本节点输出 `"123"`，类型是 `STRING`
- 数字节点输出 `123.0`，类型是 `NUMBER`

所以第一版采用“小节点、单职责、单主输出”的设计，不做一个带多输出的大常量节点。
